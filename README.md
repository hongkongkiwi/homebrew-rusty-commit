# Homebrew Tap for Rusty Commit

This is the official Homebrew tap for [rusty-commit](https://github.com/hongkongkiwi/rusty-commit), a Rust-based AI commit message generator.

## Installation

```bash
# Add the tap
brew tap hongkongkiwi/rusty-commit

# Install rusty-commit
brew install rusty-commit
```

## Usage

```bash
# Generate a commit message for staged changes
rco

# Or use the full binary name
rusty-commit
```

## Requirements

- **macOS** or **Linux**
